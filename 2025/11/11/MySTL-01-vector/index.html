std::vector

简介

​ std::vector是C++标准模板库中定义的动态数组容器，其能根据需要来动态调整存储空间大小，同时提供了与普通数组类似的随机访问性能。

​ 其定义在vector头文件中，有如下两种定义:

​ 两种方式主要是内存分配器的差异，通常使用std::vector即可。

基本特性

​ std::vector 的底层实现是一块具有连续内存的数组，其主要的基本特性如下：

  1. 动态大小：数组的大小不固定，根据需要进行变化；
  2. 连续存储：数组内部的元素在内存中是连续存储的，这也决定了其可以实现随机访问；

​ 在此提供其增删查改的时间复杂度说明表格：

操作类型 平均时间复杂度 说明
访问/修改 O(1) 连续存储，支持下标快速访问。
尾部插入 / 删除 O(1) 在触发扩容机制时需要增加重新分配内存的时间开销，平均为O(1)。
中间插入 / 删除 O(n) 需移动大量元素。
查找 O(n) 无序容器，需遍历。

内存管理与自动扩容机制

​ std::vector内存管理模型由三个指针控制：

  • begin_ 指向已分配空间的起始位置；
  • end_ 指向当前已使用空间的末尾（最后一个元素的下一个位置）；
  • end_of_storage_ 指向整个已分配空间的结尾。

​ vector数组的元素个数可以由end_-begin_计算得到，而容量大小由end_of_storage_-begin_计算得到。当数组元素个数超过了分配的容量时，将会重新申请更大的内存空间，并且将旧内存的元素移动至新内存中，并释放旧内存，同时更新内存管理指针。此外，std::vector 提供了如 reserve() 与 shrink_to_fit() 等接口，用于手动管理容量。

常见成员函数

1. 构造与赋值

函数 说明
vector() 默认构造函数，创建空容器。
vector(n) 创建含 n 个默认值元素的容器。
vector(n, const T& value) 创建含 n 个相同值元素的容器。
vector(il) 使用初始化列表构造。
vector(const vector& other) 拷贝构造函数。
vector(vector&& other) 移动构造函数。
operator= 赋值运算符。
assign(n, value) 重新赋值 n 个指定元素。
assign(initializer_list<T> il) 用初始化列表赋值。

2. 容量管理

函数 说明
size() 当前元素个数。
capacity() 当前分配的容量。
empty() 判断容器是否为空。
reserve(n) 预分配至少 n 个元素的空间。
resize(n) 改变元素数量。
shrink_to_fit() 尝试释放多余的容量。
max_size() 可容纳的最大元素数。

3. 元素访问

函数 说明
operator[](i) 访问第 i 个元素。
at(i) 访问第 i 个元素。
front() 返回首元素引用。
back() 返回尾元素引用。
data() 返回底层数组指针。

4. 修改操作

函数 说明
push_back(const T& value) 在末尾插入元素。
emplace_back(args...) 原地构造并插入尾部。
pop_back() 删除最后一个元素。
insert(pos, value) 在指定位置插入元素。
emplace(pos, args...) 在指定位置原地构造元素。
erase(pos) 删除指定位置元素。
erase(first, last) 删除指定范围的元素。
clear() 清空容器内容。
swap(vector& other) 交换两个 vector 的内容。

5. 迭代器

函数 说明
begin() / end() 返回首/尾迭代器。
cbegin() / cend() 常量版本（C++11）。
rbegin() / rend() 反向迭代器。
crbegin() / crend() 常量反向迭代器。
