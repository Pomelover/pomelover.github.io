序言

序言

​ Lua提供了许多C不擅长的机制：高级语言、动态结构、简洁、易于测试和调试等。正因为如此，Lua具有良好的安全保证，自动内存管理，简便的字符串处理功能及其他动态数据的改变。Lua不仅易于扩展，而且易于整合。

​ 以下特征是Lua特有的：

  1. 可扩展性；
  2. 简单；
  3. 高效率；
  4. 与平台无关；

Lua的使用者

​ Lua使用者分为三大类：

  1. 使用Lua嵌入到其他应用中的；
  2. 独立使用Lua的；
  3. 将Lua与C混用的；

Chunks

Chunk是一系列语句，Lua执行的每一块语句，比如一个文件或者交互模式下的每一行都是一个Chunk。

​ 交互模式下，键入的每个命令在按下回车之后立即被执行，键入文件结束符可以退出交互模式，或者调用os.exit()函数也可以退出。在交互模式下，Lua通常把每一个行当作一个Chunk，但如果一行不是一个完整的Chunk时，它会等待继续输入直到一个完整的Chunk。

全局变量

​ 全局变量不需要声明，给一个变量赋值后即创建了这个全局变量，访问一个没有初始化的全局变量也不会出错，只不过得到的结果是nil。

​ 删除一个全局变量将变量赋值为nil即可。

词法约定

  1. 标示符：字母或者下划线开头的字母、下划线、数字序列(最好不要使用下划线加大写字母的标示符)。

  2. 保留字：

    1 2 3 4 5 6 7
    and break do else elseif end false
    for function if in local nil not
    or repeat return then true until while

  3. 注释：单行注释:–,多行注释:–[[ –]]。

类型和值

​ Lua是动态类型语言，变量不需要类型定义。

​ Lua有8个基本类型：nil,boolean,number,string,userdata,function,thread,table。函数type可以测试类型。

nil

 `nil`是`Lua`中特殊的类型，全局变量没有被赋值前默认为`nil`，给全局变量赋值为`nil`，可以删除该变量。

boolean

​ 两个取值false和true。注意：**Lua中除了false和nil为假，其它均为真**。

number

​ number表示实数，Lua中没有整数。

string

​ string指字符的序列。Lua中字符串是不可以修改的，其和其它对象一样，Lua进行自动内存分配和释放，可以使用单引号或双引号表示字符串。

function

​ 函数在Lua中同样是第一类值，可以存储在变量中。

userdata and thread

​ userdata可以将C数据存放在Lua变量中，后续章节将详细介绍线程。

表达式

​ Lua中的表达式包括数字常量、字符串常量、变量、一元和二元运算符、函数调用。

算术运算符

  1. 二元运算符：+-*/^
  2. 一元运算符： -(负值)

关系运算符

赋值语句

​ 赋值是改变一个变量的值和改变表域最基本的方法。Lua可以对多个变量同时赋值，变量列表和值列表的各个元素用逗号分开，赋值语句右边的值会以此赋给左边的变量。

​ 遇到赋值语句，Lua会先计算右边所有的值然后再执行赋值操作，所以我们可以这样进行交换变量的值：

​ 当变量个数和值的个数不一致时，Lua会一直以变量个数为基础采取以下策略：

  1. 变量个数＞值的个数：按变量个数补足nil;
  2. 值的个数＞变量个数：忽略多余的值；

​ 多值赋值常用来交换变量，或将函数调用返回给变量。

局部变量与代码块

​ 使用local创建一个局部变量，与全局变量不同，局部变量只在被声明的哪个代码块内有效。代码块：指一个控制结构内，一个函数体或者一个Chunk。

尽可能的使用局部变量：

  1. 避免命名冲突；
  2. 访问局部变量速度比全局变量更快；

可以采用 do end控制局部变量作用范围。

以下重点介绍for语句：

数值for循环

​ for将用exp3作为step从exp1到exp2，执行循环体。其中如果省去exp3，则默认为1。

​ 需要注意以下几点:

  1. 三个表达式只会被计算一次，并且是在循环开始前；

  2. 控制变量var是局部变量自动被声明，并且只在循环内有效；

  3. 循环过程不要改变控制变量的值，退出循环请用break语句；

泛型for循环

​ 泛型循环与数值循环有两点相同：

  1. 控制变量是局部变量；
  2. 不要修改控制变量的值；

break和return语句

​ break用来退出当前循环，而return用来从函数返回结果，当函数自然结束结尾会有一个默认的return。

​ Lua语法要求break和return只能出现在block的结尾一句。
